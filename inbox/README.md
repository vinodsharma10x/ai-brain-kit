# Inbox

Drop anything here: notes, links, half-thoughts, screenshots, voice memo transcripts. No organizing required at capture time, that is the whole point.

Say **"process my inbox"** and Claude proposes a home for each item (a project, a person, raw/, an area), moves it after you confirm, and leaves this folder empty.

An empty inbox is a processed inbox. A full root folder is what happens to systems without one.
